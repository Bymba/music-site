Stretchy button with css

This approach has been tested with Firefox, Safari, Chrome, Opera, Internet Explorer 7 and higher.

This is an example of CSS Sliding Doors technique

This button automatically resizes width depending upon content. It was created using one image (prite)/ This button has only 2 part: left image with icon (it has fixed width) and image with right part(it's width can be changed upon content, max width is about 550 px;) To change width of the button, you don't need change related images, just add or remove text in the button
This box can be made any size without any problems with graphics. In other words, you don't have to create a large graphic for this box to accommodate extra large sizes. This box technique Validates under "HTML 4.01 Strict" and "W3C CSS". This technique W3C validates under both HTML 4.01 Strict and CSS guidlines.
